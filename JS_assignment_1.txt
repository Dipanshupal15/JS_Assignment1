Que 1- What are the different data types present in JavaScript ?
Ans 1- Primitive Data Types:
       Number : Floating point numbers for both integers and decimals.
       String : Textual data.
       Boolean : Logical data - true or false.
       Undefined : Indicates a variable that hasn't been assigned a value.
       Null : Represents the intentional absence of any object value.

       Non-primitive Data Types:
       Object : A collection of key-value pairs.
       Array : Ordered collection of values.
       Function : A reusable set of statement that we can easily call whenever needed.



Que 2- What are the JavaScript Engines ?
Ans 2- JavaScript engines are programs that execute JavaScript code.
       Some popular engines include:
       - V8 (Chrome, Node.js)
       - SpiderMonkey (Firefox)
       - Chakra (Microsoft Edge)
       - JavaScriptCore (Safari)
       - Hermes (React Native)
       - JerryScript



Que 3- Expalin ECMA in JavaScript.
Ans 3- ECMA, which stands for European Computer Manufacturers Association, is an organization responsible for 
       standardizing information and communication systems. In the context of JavaScript, ECMA refers to the
       standardization of the language specification. The official standard for JavaScript is known as ECMAScript.



Que 4- Why We Use JavaScript ?
Ans 4- JavaScript is primarily used for:

       - Enhancing Web Pages: Adding interactivity, animations, and dynamic content.
       - Web Development: Building web applications, both front-end and back-end.
       - Server-Side Development: Node.js allows JavaScript to be used for server-side scripting.

       JavaScript's History:
 
       - Birth: Developed by Brendan Eich at Netscape in 1995.
       - Early Years: Originally named LiveScript, later renamed JavaScript.
       - Standardization: ECMAScript was created to standardize JavaScript in 1997.
       - Evolution: Continuous updates and enhancements to the language (ES6, ES7, etc.) improving functionalities.

Que 5- What is DOM in JS ?
Ans 5- DOM stands for Document Object Model. In the context of JavaScript, the DOM is a programming interface that represents
       the structure of a document as   a tree of objects. This document can be an HTML or XML document, and the DOM provides
       a way for programs to manipulate the structure, style, and content of the document.
